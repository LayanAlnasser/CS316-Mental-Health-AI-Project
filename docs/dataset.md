# Project Datasets

This project uses two custom-written datasets:

---

## 1. English Mental Health Dataset

**File:** `Dataset_Ai_in_mental_health.csv`  
**Format:** CSV  

**Description:**  
This dataset contains English text entries labeled for mental health categories such as anxiety and depression.  
All entries are written manually for academic training purposes.

**Download:**  
[Click here to download](datasets/Dataset_Ai_in_mental_health.csv)

---

## 2. Arabic Mental Health Dataset

**File:** `mental_health_arabic_dataset_new.xlsx`  
**Format:** Excel (XLSX)

**Description:**  
This dataset contains Arabic text entries related to mental health with associated labels.  
It was written manually to support multilingual NLP experiments.

**Download:**  
[Click here to download](datasets/mental_health_arabic_dataset_new.xlsx)

---

### Important Notes

- These datasets were fully authored for academic use within this project.
- No personal information or external user data is included.
